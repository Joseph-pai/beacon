# APP Beacon Taiwan

APP Beacon 台灣官方網站 - 專業 APP 代理開發服務

## 功能特色

- 🚀 一站式 APP 代理開發服務
- 📁 代碼庫管理系統
- 🎨 響應式設計，支持多種設備
- ✨ 粒子動畫背景

## 技術堆疊

- HTML5
- CSS3
- JavaScript
- LocalStorage (數據持久化)
- JSZip (打包下載)

## 開發

```bash
# 啟動本地服務器
python3 -m http.server 8000
```

## 部署

本項目已配置 Netlify 自動部署，推送至 GitHub 後自動構建。
